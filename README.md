## 功能特点

- 🚀 快速访问剪贴板历史
- 📝 支持文本和图片内容
- 🏷️ 标签分类管理
- 🔍 内容快速搜索
- 🌓 自适应明暗主题
- ⌨️ 快捷键支持
- 💾 本地数据持久化

## 快捷键

- `Alt + C` 或 `Shift + Alt + V`: 显示/隐藏主窗口
- `/`: 快速搜索
- `←/→`: 在历史记录中导航
- `Enter`: 复制选中内容
- `Delete/Backspace`: 删除选中记录

### 环境要求

- Go 1.18+
- Node.js 16+
- Wails v2

### 安装指南

```
git clone https://github.com/JJ-H/clipboard.git

cd cliboard

make build

```
